# `flatten`

Makes any matrix 1 dimensional.

---

## Syntax

#### `flatten(matrix: any[] | Mat | Tensor) : any[]`

**param** `matrix` - an `Array`, `Mat` or `Tensor`

**returns:** `any[]` - A list with all elements from `matrix`. The order of elements follows the dimensions or shape order. 

---
