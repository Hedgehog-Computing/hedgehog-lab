# `inv`

Inverse of a matrix.

---

## Syntax

#### `inv(matrix: any[] | Mat) : any[]`

**param** `matrix` a 2-D `Array` or `Mat`. Should be square and invertible. 

**returns:** `any[]` - The matrix that is the *inverse* of `matrix` i.e. `matrix * matrix^-1 = I` where `I` is the identity matrix. (recall identity matrix is 1's on the diagonal and 0's elsewhere).

---

## Description

This function simply finds the inverse of a matrix if it's invertible. One test for invertible matrices is its invertible if the determinant is not zero. ( `!(det(matrix) === 0` )
