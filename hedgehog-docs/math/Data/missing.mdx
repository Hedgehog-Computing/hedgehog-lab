# `missing`

Adds extra missing space.

---

## Syntax

#### `missing(matrix: any[] | Mat | Tensor, dim: number, times: number) : any[] | Mat | Tensor`

**param** `matrix` - The input matrix, `Tensor`, `Mat`, etc that gets added missing space to it

**param** `dim` - '0' or '1' - 0 means row and 1 means column. This is where you create the extra space

**param** `times` - how many rows / columns to create 

**returns:** `any[] | Mat | Tensor` - A copy of the input with extra blank (**missing**) rows or columns (based on parameter `dim` with the amount based on parameter `times`).

---

## Description

This function creates a copy of your input with extra space, which is missing space. 

Note that `null`, `NaN`, `''` and `undefined` are defined to be missing values. `[]` and `{}` are not. 

This is useful in combination with the function `fill_missing`.
