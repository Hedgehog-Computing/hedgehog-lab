# `all`

Imports all functions from the math library.

---

## Syntax

#### `all()`

**param** none

**returns** none

---

## Description

This function is useful if you don't want to handpick what functions to import. It *imports all math functions* from the math library (including all discussed in this reference) into the Hedgehog Lab instance.
