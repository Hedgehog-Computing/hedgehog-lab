# Output

Output is crucial to any programming language. They define how a user can access information in different ways. There's a few key ways to do so:

There is the browser console using `console.log()` along with `print()` in Hedgehog Script. The first logs things to the web browser console, while the latter prints it out in the Hedgehog Lab environment. 

:::tip
In Chrome use 'F12' to open the console. Other browsers may have a different, yet simple, method.
:::
