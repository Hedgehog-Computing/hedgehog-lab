# Operators & Expressions

There are infinitely many operators and expressions. However, JavaScript has built-in ones and that's what we'll cover here. First let's go over what a expression and operator is, exactly. An expression is a "valid set of literals, variables, operators and expressions that evaluate to a single value that is an expression". That value can be a string, number, boolean, etc. Operators are "special symbols used to perform operations on operands (values and variables)". Let's go over all of them:
