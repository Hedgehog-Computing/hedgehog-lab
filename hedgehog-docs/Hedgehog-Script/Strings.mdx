#  Strings

All properties and features from the `String` object in JavaScript/TypeScript are also available in Hedgehog Script. You can assign strings to variables and they can also be treated like arrays. See JavaScript documentation for more: 

https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String
