# Contribution 

Thank you for reading this book!

If you wish to contribute to this book, here's our process:

- 1) Fork the repository on [Hedgehog Book Github](https://github.com/Hedgehog-Computing/docs)

- 2) Make sure to have Yarn installed

- 3) In the terminal, run `yarn` and then `yarn start` (debug mode)

- 4) Contribute in the way you'd like

- 5) Submit for merging

A quick guide on certain features:

- MDX is used along with Docusaurus, for extra resources visit their sites

- To add pictures, put said picture in `.../static/img/{pictureFileName}`
    - To display, write: `![Description](/img/{pictureFileName})`

- To add code blocks start one with ` ```js ` , and end it with ` ``` `

- For inline code simply wrap said code in backticks (`)

**Adding a code snippet:**

Add an iframe element with the source being your snippet link:

-<iframe
    src="snippet_link"
    width="100%"
    height="600px"\>

**Building a static site:**

- Run `yarn build` in your terminal
