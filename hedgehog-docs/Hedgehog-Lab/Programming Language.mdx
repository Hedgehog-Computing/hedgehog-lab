# The Programming Language

1. The programming language used in Hedgehog Lab is **Hedgehog Script**

2. **Hedgehog Script** is a high level dynamic programming langauage compatible with and built on **JavaScript/TypeScript**

3. It compiles and executed inside your browser

4. It is **multi-paradigm**
    - Functional Programming (FP)
    - Object-Oriented Programming (OOP)
    - Dynamic types
    - Metaprogramming

5. Sweet syntax for **Matrix** and **Tensor** operations

6. Easy to import packages/modules using `*import`

:::info

Our book on Hedgehog Script can be found via: [Hedgehog Script](../Hedgehog-Script/intro-hedgehog-script)

:::
