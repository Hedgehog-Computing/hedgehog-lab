## Features of Hedgehog Lab

**Now that we've registered, in this section we will display the various buttons that provide features in Hedgehog Lab which include:**
<ul>
<li>The Sidebar</li>
<li>Account & Profile</li>
<li>Resources</li>
<li>The Programming Environment</li>
</ul>
We will use an array tutorial example for the visual reference: 

![features](/img/features.png)

Let's discuss the various features in hedgehog lab:

<ul>
<li>Opening the sidebar: simply click the top-left 3 horizontal bars. This gives access to:
    <ul>
    <li>New snippets: create a new snippet of code</li>
    <li>Explore: look at various recent, or search for snippets of public code</li>
    <li>Timeline: look at a chronological timeline of the snippets added according to who you follow </li>
    <li>Current: the current snippets you're working on</li>
    <li>My profile: view your profile information </li>
    </ul>
    </li>
<li>Top right: resources & account
<ul>
<li>Book: what you're reading now! The documentation on Hedgehog Lab in a book format </li>
<li>Discord: our official discord for Hedgehog Lab</li>
<li>Github: our page on github along with the number of stars we have </li>
<li>Account: another way to access and change your basic profile information</li>s
</ul>
</li>
    
<li>Draft: Continue working on the last snippet you were working on, if not finished</li>
<li>Getting started with examples: Some filled in snippets giving an idea of how to write code</li>
<li>From the community: Another way to see various snippets that have been added publically</li>
</ul>

Now that we've familiarized ourselves with the features provided by Hedgehog Lab, click "New snippets" and enter the environment of hedgehog lab!
