# The Environment

1.  The **left-side bar** manages login information along with all your code snippets

2. The **codepad** (left-hand side window) is your input for source code

3. The **output** is on the right and it contains all execution results

4. The **run** button causes code to execute (if not in **LiveMode**)

5. The **LiveMode** switch is at the center top and when on, executes continuously.
